MakeStyle.html: Creating Html Styles

This page provides a program to create and test the attributes of container tags,
the common ones being

  • Headers: <hn> </hn> pairs where n=1, 2, 3, 4, 5
  • Indents and lists: <p> </p>, <ul> </ul>, <li> </li>
  • Containers: <body> </body>, <div> </div>, <span> </span>
  • Tables: <table> </table>, <tr> </tr>, <th> </th>, <td> </td>

To Create and Test a Style
  1. Go to Table 1
  2. Empty all parmeters not required
  3. Provide values for all parameters required
  4. At the bottom of table 1, click Copy button to one of the rows
  5. Check the results in Table 2
  6. Correct style in text box in Table 2 and re-test result using Action button
  7. The style is automatically copied to clipboard
  8. Paste style in clipboard to html files
